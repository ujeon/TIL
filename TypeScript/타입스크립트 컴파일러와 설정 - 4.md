`tsconfig.json` 파일의 `"compilerOptions"` 에는 설정할 수 있는 옵션과 옵션에 대한 짧막한 설명이 담겨 있습니다. 여기서는 이  많은 옵션들 중에서 `"target"` 옵션에 대해서 알아보겠습니다.

기본적으로 `"target"` 옵션은 `"es5"` 로 설정되어 있습니다.

```json
{
  "compilerOptions": {
    /* Basic Options */
    "target": "es5",                          /* Specify ECMAScript target version: 'ES3' (default), 'ES5', 'ES2015', 'ES2016', 'ES2017','ES2018' or 'ESNEXT'. */
    "module": "commonjs",                     /* Specify module code generation: 'none', 'commonjs', 'amd', 'system', 'umd', 'es2015', or 'ESNext'. */
    // ... 생략
```

`"es5"`로 설정하면 타입스크립트를 컴파일 할 때 `"es5"` 문법을 사용하여 컴파일 하게 됩니다. 예를 들어 `const`나 `let` 대신 `var` 를 사용하는 셈이죠. `"target"`을 여러분이 원하시는 자바스크립트 버전으로 바꾸시면 됩니다!
